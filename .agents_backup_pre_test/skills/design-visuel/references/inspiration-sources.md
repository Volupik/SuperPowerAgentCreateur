# Ressources & Inspiration Design

Une liste curée de sites de référence pour trouver de l'inspiration visuelle de haute qualité avant de commencer un projet. Ces sources proposent des designs modernes qui évitent les clichés génériques.

## Héro Sections & Landing Pages
- **[Supahero](https://supahero.io/)** : Superbe curation de hero sections premium, claires et impactantes. Excellente source pour l' "Above The Fold".
- **[Lapa Ninja](https://www.lapa.ninja/)** : Galerie très complète de landing pages entières, souvent très illustrées et colorées.
- **[Landingfolio](https://www.landingfolio.com/)** : Excellente collection de pages SaaS, Web3 et startups tech.
- **[One Page Love](https://onepagelove.com/)** : Le temple des sites "One Page", parfait pour les portfolios, apps, et lancements de petits produits.

## Design Produit / SaaS / Web App
- **[Mobbin](https://mobbin.com/)** : Capture intégrale des flux (UI complète) des applications iOS, Android et Web les plus populaires du monde. La meilleure source pour l'UX et les dashboards SaaS.
- **[UI Garage](https://uigarage.net/)** : Patterns UI spécifiques triés par catégorie (login, pricing, dashboard, etc.).
- **[SaaS Interface](https://saasinterface.com/)** : Galerie d'inspiration UI pour le logiciel B2B et SaaS.

## Excellence Générale & Avant-Garde
- **[Awwwards](https://www.awwwards.com/)** : Le standard de l'industrie pour le web design primé. Souvent complexe à implémenter, mais excellent pour l'inspiration d'animations, typographies gigantesques et layouts "cassés".
- **[Godly Website](https://godly.website/)** : Les meilleurs sites web avec animations premium (scroll réactif, WebGL, typographie avant-gardiste).
- **[SiteInspire](https://www.siteinspire.com/)** : Curation très élégante et sobre. Style de design très éditorial, typographique et "clean". Très bien pour l'e-commerce luxe, l'art, les agences d'architecture.
- **[Dark Mode Design](https://www.darkmodedesign.com/)** : Uniquement dédié aux sites web en dark mode remarquablement designés.

## Éléments Détaillés (Couleurs, Typo)
- **[Typewolf](https://www.typewolf.com/)** : Quelle police est utilisée sur ce beau site ? Les meilleures paires de polices du web. (Pour éviter Inter/Roboto).
- **[Fonts In Use](https://fontsinuse.com/)** : Archive indépendante de design graphique et web indexée par typographie.
- **[Huemint](https://huemint.com/)** : Générateur de palette de couleurs par machine learning qui applique intelligemment les couleurs d'accentuation sur des maquettes.

## Pourquoi Scrapper l'Inspiration ?
L'IA a tendance à générer du design "moyen" car elle est entraînée sur des millions de sites moyens. Les sites listés ci-dessus appartiennent aux 5% supérieurs de l'industrie du design. Analyser ces sites permet de "spiker" la créativité et de générer un design humain et distinctif.
