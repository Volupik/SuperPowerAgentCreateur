## 1. Recherche de Mots-Clés

### Méthodologie complète
*   **Étape 1 : Brainstorm initial** (Identifier les thèmes, problèmes résolus, vocabulaire client)
*   **Étape 2 : Expansion avec outils** (Google Keyword Planner, Ahrefs, SEMrush, PAA)
*   **Étape 3 : Qualification** (Volume, KD, Intention, Conversion, Pertinence)
*   **Étape 4 : Regroupement** (Grouper par intention et par topic cluster)
*   **Étape 5 : Priorisation** (Prioriser les quick wins)

### Analyse de la SERP
Pour chaque mot-clé cible, analyser :
1. Type de résultats affichés (Featured snippet, PAA, Pack Local, Shopping, Video)
2. Profil des pages qui rankent (Type domaine, longueur contenu, DR/DA)
3. Gaps et opportunités (Angle non couvert, format manquant)

## 2. Optimisation des Balises

### Title Tag — Formules éprouvées (Facteur #1)
Longueur : 50-60 caractères (max 580px en desktop)
*   **Page produit** : `[Nom Produit] — [Bénéfice Principal] | [Marque]`
*   **Page catégorie** : `[Mot-clé Collection] — [Différenciateur] | [Marque]`
*   **Article de blog** : `[Mot-clé Principal] : [Promesse] ([Année])`
*   **Page locale** : `[Service] à [Ville] — [USP] | [Marque]`

### Meta Description — Formules éprouvées
Longueur : 150-160 caractères
*   **Article** : `[Question/Problème] ? Découvrez [promesse de valeur]. [Résumé 1 phrase].`
*   **Local** : `[Service] à [Ville]. [USP 1], [USP 2]. Devis gratuit. Appelez le [numéro].`

## 3. Rédaction SEO Avancée

### Structure de contenu optimale (articles)
*   **H1** : Contient le mot-clé principal (Un seul par page)
*   **INTRODUCTION** : Accroche, contexte, mot-clé dans les 100 premiers mots.
*   **H2 / H3** : Sections logiques basées sur des mot-clés secondaires, questions (PAA).
*   **FAQ** : Questions fréquentes sur [Sujet].
*   **CONCLUSION** : Résumé, CTA, maillage interne.

> **Règle d'or** : La structure Hn doit former un sommaire compréhensible sans lire le texte.

### Optimisation sémantique
Couvrir le champ sémantique complet (entités liées, synonymes).
Répondre aux questions PAA avec concision.

## 4. Cannibalisation de Mots-Clés

La cannibalisation = deux pages ou plus du même site se font concurrence sur le même mot-clé. (A éviter !)
*   **Fusionner les contenus (301)** : Meilleur si même sujet exact.
*   **Différencier les intentions** : C'est-à-dire cibler des requêtes/intentions de recherche différentes.
*   **Canonical** : Si pages structurellement similaires.

## 5. Optimisation des Images Avancée

*   **Format** : `WebP` ou `AVIF` (avec fallback si besoin)
*   **Compression** : Qualité 80-85%
*   **Alt text** : Descriptif, naturel, inclut mot-clé si pertinent.
*   **Nom de fichier** : Descriptif avec tirets (`plombier-paris-15.webp`)
*   **Attributs fixes** : `width` et `height` indispensables (évite le CLS).
*   **Lazy Loading** : `loading="lazy"` sauf pour le haut de page (above-the-fold).

## 6. People Also Ask (PAA) & Featured Snippets

*   Identifier les PAA pertinentes pour le mot-clé.
*   Les intégrer comme H2/H3 ou dans une FAQ (marquée avec Schema FAQPage si éligible).
*   Répondre de manière concise (40-60 mots au format "*[Mot-clé] est...*") juste en dessous du titre. Google adore extraire ces petits paragraphes de définition !
